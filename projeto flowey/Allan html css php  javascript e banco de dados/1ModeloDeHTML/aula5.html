Tag h1 até h6

Essas tags são usadas para destacar parte de um texto, como seus títulos e sub-títulos.
Esse texto está dentro da tag < p > de parágrafo, que permite escrever uma grande quantidade
de palavras e linhas de uma única vez.


Obseserve que há um espaço entre o texto acima e esse aqui, conseguimos isso através da tag < br >,
que quebra o texto e começa em uma nova linha sem a necessidade de abertura de um novo parágrafo.

Nesse parágrafo vamos demonstrar a utilização da tag < q >, como citações.
Só sei que nada sei, essa frase famosa está dentro de aspas, demonstrando que o conteúdo não é do interlocutor original e sim uma citação.
Não é necessário utilizar o < blockquote > dentro de um bloco de parágrafo, a sua vantagem é que podemos já utilizar apenas no < body >.
